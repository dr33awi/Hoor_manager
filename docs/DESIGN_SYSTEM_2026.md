## 🎯 فلسفة التصميم

### المبادئ الأساسية

| المبدأ | الوصف | التطبيق |
|--------|-------|---------|
| **Clarity First** | الوضوح فوق كل شيء | الأرقام المالية واضحة دائماً |
| **Efficiency** | أقل نقرات ممكنة | أي عملية يومية ≤ 3 نقرات |
| **Contextual** | المعلومات المناسبة في الوقت المناسب | Adaptive UI |
| **Trustworthy** | يوحي بالموثوقية والأمان | ألوان رسمية، مزامنة مرئية |

### الفئة المستهدفة

```
┌─────────────────┬─────────────────┬─────────────────┐
│   محاسبين       │  أصحاب أعمال    │   مدراء         │
├─────────────────┼─────────────────┼─────────────────┤
│ دقة في الأرقام  │ نظرة عامة سريعة │ تقارير تفصيلية  │
│ إدخال سريع     │ متابعة يومية    │ مقارنات         │
│ تقارير مفصلة   │ تنبيهات ذكية    │ تحليلات         │
└─────────────────┴─────────────────┴─────────────────┘
```

## 🗂️ Information Architecture

### الهيكل الرئيسي

```

│
├── 🏠 Dashboard (الرئيسية)
```

## 🎨 Design Tokens

### Color Palette

#### Primary Colors (الألوان الأساسية)

```dart
// Deep Navy - اللون الرئيسي
primary:        #0F172A  // Background, Headers
primaryLight:   #1E293B  // Hover states
primarySoft:    #334155  // Secondary elements

// Ocean Blue - اللون الثانوي
secondary:      #0EA5E9  // CTAs, Links
secondaryLight: #38BDF8  // Hover
secondaryMuted: #E0F2FE  // Backgrounds
```

#### Semantic Colors (ألوان دلالية)

```dart
// Success / Income (الدخل)
income:         #10B981
incomeLight:    #D1FAE5
incomeDark:     #059669

// Error / Expense (المصروفات)  
expense:        #EF4444
expenseLight:   #FEE2E2
expenseDark:    #DC2626

// Warning (تحذير)
warning:        #F59E0B
warningLight:   #FEF3C7

// Info (معلومات)
info:           #3B82F6
infoLight:      #DBEAFE
```

#### Surface Colors (أسطح)

```dart
background:     #F8FAFC  // خلفية التطبيق
surface:        #FFFFFF  // البطاقات
surfaceMuted:   #F1F5F9  // خلفية ثانوية
surfaceElevated:#FFFFFF  // عناصر مرتفعة
```

#### Text Colors (النصوص)

```dart
textPrimary:    #0F172A  // العناوين
textSecondary:  #475569  // نص ثانوي
textTertiary:   #94A3B8  // تلميحات
textDisabled:   #CBD5E1  // معطل
```

## 📝 Typography

### Font Family

```dart
// Primary: Cairo (للعربية والإنجليزية)
// Monospace: JetBrains Mono (للأرقام المالية)
```

### Type Scale

| Style | Size | Weight | Line Height | Usage |
|-------|------|--------|-------------|-------|
| Display Large | 48sp | Bold | 1.2 | Hero sections |
| Display Medium | 40sp | Bold | 1.2 | Page titles |
| Display Small | 32sp | SemiBold | 1.25 | Section headers |
| Headline Large | 28sp | Bold | 1.3 | Card titles |
| Headline Medium | 24sp | SemiBold | 1.3 | Subsections |
| Title Large | 18sp | SemiBold | 1.4 | List items |
| Title Medium | 16sp | SemiBold | 1.4 | Labels |
| Body Large | 16sp | Regular | 1.5 | Primary text |
| Body Medium | 14sp | Regular | 1.5 | Secondary text |
| Body Small | 12sp | Regular | 1.4 | Captions |
| **Money Large** | 28sp | Bold | 1.2 | Large amounts |
| **Money Medium** | 20sp | SemiBold | 1.3 | Normal amounts |
| **Money Small** | 14sp | Medium | 1.4 | Small amounts |
